CSS Combinators and Specificity

CSS Selectors

  • In CSS, selectors are patterns used to select the elements to be styled
  • Elements can be selected by tag, class, id, attribute
  • Elements can also be targeted through the pseudo-class and pseudo-element selectors or using the universal selector
  • An element can be accessed through a combination of selectors

Selector list

  • The selector list is a group of selectors separated by comma
  • It is not considered to be a combinator
  • The same rules apply to all elements that match at least one of the selectors
h1, h2, h3 {
  color: fuchsia;
}

Multiple selectors

  • Selects elements that match all of the selectors listed
  • Not considered to be a combinator
.tab {
  text-transform: uppercase;
}
.tab.active {
  border-bottom: 2px solid fuchsia;
}

Descendant combinator

  • Combinator selectors combine two selectors
  • The descendant combinator is a white space
  • Selects any element matching the second selector that is a descendant of an element matching the first selector
<div class=“articles”>
  <article>
    <h1>…</h1>
     <p>…</p>
    …
  </article>
    …
</div>

.articles h1 {
  font-size: 4rem;
}

Child combinator

  • The direct child combinator is >
  • Selects any element matching the second selector that is a direct child of an element matching the first selector
  • Stricter than the descendant selector
<div class=“articles”>
  <article>
    <h1>…</h1>
     <p>…</p>
    …
  </article>
    …
</div>

article > h1 {
  font-size: 4rem;
}

Adjacent sibling combinator

  • The adjacent sibling combinator is +
  • Selects any element matching the second selector that is the next sibling of an element matching the first selector
<div class=“articles”>
  <article>
    <img src="…">
    <h1>…</h1>
    <p>…</p>
     <p>…</p>
    …
  </article>
    …
</div>

h1 + p {
  color: grey;
}

General sibling combinator

  • The general sibling combinator is ~
  • Selects any element matching the second selector that is one of the next siblings of an element matching the first selector
<div class=“articles”>
  <article>
    <img src="…">
    <h1>…</h1>
    <p>…</p>
     <p>…</p>
     …
  </article>
    …
</div>

h1 ~ p {
  color: grey;
}

Lobotomized owl example

  • Use case: create vertical spacing between stacked elements
  • Solution: attach margin-top to each element that follows another element
  • Lobotomized owl: * + * { }
  • Source: A List Apart

Specificity

  • Specificity is the weight that is applied to a given CSS declaration
  • CSS calculates specificity based on the selectors used in the declaration block
  • Each type of selector has a given weight
    • 0: universal selector and inherited values
    • 1: tag selector and pseudo-element selector
    • 10: class and pseudo-class selector
    • 100: id selector
    • 1000: inline styles
    • 10000: !important
