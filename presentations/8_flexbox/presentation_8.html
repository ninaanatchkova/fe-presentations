Flexbox

What is flexbox?

  • The Flexible layout module aims at providing a more efficient way to lay out, align and distribute space among items in a container
  • The elements can be of unknown or dynamic size
  • A flex layout consists of a parent element — the flex container — and its children — the flex items
  • The flex layout gives the flex container the ability to alter its flex items's width, height and order to best fill the available space
  • The flex layout is direction agnostic, as opposed to the block layout (vertically-based, like a column) and the inline layout (horizontally-based, like a row)

Terminology

  • Unlike CSS Grid and other grid-based systems, the flex layout can be used to lay out items only along one dimension (row or column). It is a one-dimensional layout model.
  • To achieve this, one of the two dimensions of the flex container becomes its main axis and the other dimension takes on a support role as its cross axis.

Main axis

  • The main axis of the container is the primary axis along which the flex items are laid out
  • The main axis has a direction, which is delineated by the main start and main end of the container. By default it extends from the left side of the container to right.
  • The direction of the main axis can be changed using the flex-direction property from row (horizontal) to column (vertical). Other options are row-reverse and column-reverse.

  • Main start and main end are the starting and ending point of the container along the main axis. The flex items are placed along the main axis starting at main start and ending at main end.
  • The main size is the dimension of the container along the main axis. It could be the width or the height of the container, depending on the flex-direction of the container.

Cross axis

  • The cross axis is the axis perpendicular to the main axis of the flex container.
  • The cross start is the top side of the container for flex-direction: row and flex-direction: row-reverse and the left side of the container for flex-direction: column and flex-direction: column-reverse.
  • The cross size of the container is the dimension other than the dimension which is its main size.

The flex container

The display property

  • To turn an element into a flex container, its display property should be set to flex or inline-flex. The inline-flex container behaves similarly to an inline-block element in the document flow, while the flex container behaves like a block level element.
    •           
            .container {
              display: flex; /* or inline-flex */
            }
  • When an element is converted to a flex container, all of its children (which have not been removed from the normal document flow) automatically become flex items.
  • The defaults applied to a flex container and its children are:
    • The flex items display in a row (flex-direction: row)
    • The flex items start from the start edge (main start) of the main axis
    • The flex items do not stretch to fill the length of the main dimension, but they can shrink to fit in it
    • The flex items stretch to fill the size of the cross axis.
    • The initial size of the flex items is just as much as is required to fit the content (flex-basis: auto)
    • All flex items are forced onto one line (flex-wrap: nowrap)

Flex direction

  • The flex-direction property establishes the main axis of the flex container
  • The directions of row and row-reverse depend on the direction of the HTML element (left-to-right: ltr or right-to-left: rtl). This can be set through the dir HTML attribute.
          
            .container {
              flex-direction: row | row-reverse | column | column-reverse;
            }

Flex wrap

  • By default, the flex items are forced to fit onto one line
  • They can be allowed to wrap with the wrap or wrap-reverse options
          
            .container {
              flex-wrap: nowrap | wrap | wrap-reverse;
            }

Flex flow

  • Shorthand for flex-direction and flex-wrap
          
            .default-container {
              flex-flow: row nowrap;
            }
            .custom-container {
              flex-flow: column wrap-reverse;
            }

Justifying content along the main axis

  • Used to distribute the space between the elements in the flex container
        
          .container {
            justify-content: flex-start | flex-end | center |
            space-between | space-around | space-evenly;
          }
