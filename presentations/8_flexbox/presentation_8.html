Flexbox

What is flexbox?

  • The Flexible layout module aims at providing a more efficient way to lay out, align and distribute space among items in a container
  • The elements can be of unknown or dynamic size
  • A flex layout consists of a parent element — the flex container — and its children — the flex items
  • The flex layout gives the flex container the ability to alter its flex items's width, height and order to best fill the available space
  • The flex layout is direction agnostic, as opposed to the block layout (vertically-based) and the inline layout (horizontally-based)
